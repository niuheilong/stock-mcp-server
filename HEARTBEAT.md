# HEARTBEAT.md - 系统健康监控

## 自检任务（每次心跳执行）

### 1. 会话健康检查
```bash
# 检查内存文件大小
find ~/.openclaw/workspace/memory -name "*.md" -size +1M -exec ls -lh {} \;

# 如果超过阈值，自动归档并告警
```

### 2. 决策规则
- **文件 > 5MB**: 强制归档，开启新会话
- **消息堆积 > 10条未回复**: 静默等待，不主动发送
- **连续错误 > 3次**: 停止自动重试，报告用户

### 3. 长任务策略
- 复杂任务 → 使用 `sessions_spawn` 创建子会话
- 文件处理 → 流式读取，避免一次性加载
- 循环操作 → 设置最大迭代次数，超限则中断
